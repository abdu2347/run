# 快速部署指南

## 当前版本

这是一个**纯前端**应用，无需Python环境，无需任何插件！

## 部署步骤

### 第一步：准备文件

1. 确保 `wx.jpg`（你的微信二维码）放在 `run` 文件夹中
2. 确保 `index.html` 在 `run` 文件夹中

### 第二步：选择部署方式

#### 推荐：使用 Vercel（5分钟完成）

1. 访问 https://vercel.com 并注册账号
2. 点击 "New Project" → "Import Git Repository" 或直接上传
3. 将 `run` 文件夹内容拖入
4. 自动部署，获得免费域名

#### 备选：使用 Netlify

1. 访问 https://app.netlify.com/drop
2. 直接拖入 `run` 文件夹
3. 获得免费临时域名，可绑定自己的域名

### 第三步：绑定自定义域名

在 Vercel 或 Netlify 的项目设置中：
1. 进入 Domains 设置
2. 添加你的域名
3. 按提示添加 DNS 记录

## 部署后的效果

- 任何设备（手机、电脑、平板）访问域名即可使用
- 无需安装任何APP或插件
- 微信二维码和联系方式显示在页面底部
